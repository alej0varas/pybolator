pybolator
=========

PyBoard emulator

Run
---

Put your `main.py` in the project root and run:

  $ python3 pybulator.py

Development
-----------

  $ PYBOLATOR_MAIN=test.py python pybolator.py

Dependencies
------------

`python3-tk`
````````````
Debian based distros:

  $ apt-get install python3-tk
